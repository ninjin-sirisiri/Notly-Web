Notly 機能一覧
================

Notly は「惰性に負けず、毎日ノートが書ける」ことに特化した、シンプルで高速なデスクトップノートアプリです。下記のセクション構成は、そのまま Web サイトのセクションとして利用できるように設計されています。

## 1. コアコンセプト

- **習慣化を支えるノートアプリ**  
  ノートを「書きたいときだけ書くもの」ではなく、「毎日続けられる習慣」としてデザイン。
- **オフラインで完結するプライベート空間**  
  データはローカルに保存され、クラウド連携なしでも安心して書き溜められます。
- **軽量・高速なネイティブ体験**  
  Tauri + SQLite によるデスクトップアプリとして、ブラウザ依存の Web アプリよりも軽快に動作します。

## 2. ノート作成・編集

- **Markdown ベースのリッチエディタ**  
  Tiptap ベースのエディタで、見出し・箇条書き・コードブロックなど、Markdown 中心の表現に最適化。
- **書くことに集中できるミニマル UI**  
  余計なパネルを排し、エディタと必要最小限のツールバーだけに絞ったレイアウト。
- **リアルタイム自動保存**  
  入力のたびにローカルへ保存され、編集中のクラッシュや誤操作によるロストを最小化。
- **ノートの更新履歴を意識したタイムスタンプ**  
  作成日時・更新日時を明示し、「いつ書いたか」「いつ見直したか」がすぐ分かります。

## 3. フォルダ & ファイル管理

- **階層フォルダでノートを整理**  
  フォルダを自由に作成し、階層構造でノートを整理可能。
- **ドラッグ & ドロップで移動**  
  ノートやフォルダをドラッグして、直感的に階層を組み替えられます。
- **フォルダごとのソート設定**  
  名前順 / 作成日 / 更新日など、フォルダごとに並び替えルールをカスタマイズ可能。
- **色・アイコンでフォルダを視覚的に区別**  
  よく使うフォルダやプロジェクト単位のフォルダに色やアイコンを設定し、パッと見で判別できます。

## 4. 検索 & 絞り込み

- **全文検索対応のノート検索**  
  タイトルだけでなく本文まで含めた検索で、キーワードから目的のノートを即座に特定。
- **階層構造を保ったままの検索結果表示**  
  一致したノートだけでなく、その親フォルダも自動的に展開して見つけやすく表示。
- **タグによる絞り込み（実装中 / 一部対応）**  
  特定のタグが付いたノートだけを抽出して一覧化。プロジェクトやテーマごとに横断的な整理が可能になります。
- **並び替えとフィルタの組み合わせ**  
  検索結果も含めて、「名前」「作成日」「更新日」などの軸でソートし直せます。

## 5. 習慣化サポート機能

- **連続日数（ストリーク）の記録**  
  ノートを作成・更新した日を自動で記録し、「何日連続で書けているか」を可視化。
- **「今日書いた」を実感できるフィードバック**  
  新しくノートを書いたり、既存ノートを更新すると、ストリークが更新され、小さな達成感を積み重ねられます。
- **将来の拡張を見据えた習慣機能**  
  通知やリマインダー、週間・月間の振り返りダッシュボードなどの機能追加を前提にした設計。

## 6. テンプレート & クイックノート

- **ノートテンプレート機能（実装中 / 一部対応）**  
  よく使うフォーマット（日報・振り返り・会議メモなど）をテンプレートとして登録し、ワンクリックで新規ノートを作成。
- **テンプレートエディタ**  
  テンプレート自体も Markdown ベースで編集でき、細かい表現を自由にカスタマイズ可能。
- **クイックノート用のショートカット連携**  
  グローバルショートカットから即座にノート作成画面を開き、「思いついた瞬間」のメモを逃しません。

## 7. ショートカット & 生産性

- **グローバルショートカット対応**  
  アプリがバックグラウンドでも、キーボードショートカットからすぐに Notly を呼び出し可能。
- **ショートカットのカスタマイズ**  
  「クイックノートを開く」「ウィンドウの表示／非表示を切り替える」などのアクションごとに、好みのキー割り当てを設定できます。
- **プラットフォームに合わせたキー表記**  
  macOS では Cmd / Option、Windows では Ctrl / Alt など、利用環境に合わせた見やすい表記に自動変換。

## 8. ゴミ箱 & 安全な削除

- **ノート・フォルダの論理削除（ゴミ箱）**  
  誤って削除してもすぐには消えず、ゴミ箱から復元できます。
- **完全削除もワンクリックで**  
  不要なノートやフォルダは、ゴミ箱から完全削除することでストレージを整理可能。

## 9. プライバシー & データ管理

- **すべてローカル保存**  
  ノートデータは SQLite データベースとしてローカルに保存。クラウドサーバーに送信されないため、機密メモも安心です。
- **ファイルシステムベースの構造**  
  実際のディレクトリ構造と対応したフォルダ管理により、バックアップや移行もしやすい設計。
- **今後のバックアップ機能拡張を想定**  
  外部ドライブやクラウドストレージへのエクスポート／インポート機能など、将来的な拡張を見据えた構成になっています。

## 10. UI デザイン & 体験

- **モダンでプレミアムな UI**  
  Glassmorphism を取り入れた半透明レイヤーや、控えめなシャドウ・グラデーションで、書く体験を邪魔しない「静かなプレミアム感」を演出。
- **スムーズなアニメーション**  
  フォルダの展開やモーダル表示など、操作に対するアニメーションを最小限・高速にまとめ、気持ち良いレスポンスを実現。
- **ダークモード対応（予定）**  
  長時間の作業でも目に優しいダークテーマを今後提供予定です。

## 11. 技術スタック（開発者向け）

- **フロントエンド**  
  - React (Vite) / TypeScript  
  - Tailwind CSS v4 + shadcn/ui によるモダンなコンポーネント設計  
  - Zustand による軽量な状態管理  
  - Lucide React アイコン / Tiptap ベースエディタ
- **バックエンド / ネイティブ**  
  - Tauri v2（軽量・高速なデスクトップアプリフレームワーク）  
  - Rust による安全で高パフォーマンスな実装  
  - SQLite によるローカルデータベース
- **開発ツール**  
  - Bun ベースの開発環境  
  - Oxlint / Oxfmt による厳密な Lint / Format チェック
